# Installation

1. Установить зависимости.

```bash
bundle i
```
2. Настроить соединение с базой: config/database.yml

3. Применить миграцию

```bash
rails db:migrate
```

3. Заполнить тестовыми данными

```bash
rails db:seed
```

3. Запустить сервер

```bash
rails s
```
